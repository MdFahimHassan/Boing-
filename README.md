# Boing! 🏀

Boing! is a nostalgic web revival inspired by classic bouncing ball adventures. Built from scratch as a web app, it combines retro charm with modern polish. Simple controls, addictive physics, and vibrant design make every bounce a blast.

---

## 🚀 Project Goals
- Recreate the fun of Bounce Tales with a fresh web experience.
- Deliver a minimal playable prototype in Week 1.
- Expand with multiple levels, scoring, and polish in later sprints.

---

## 👥 Team Roles
- **UI/UX (Fahim)**: Menus, HUD, styling, branding.
- **Game Logic (Tamzid)**: Physics, collisions, scoring system.
- **Level Design (Prithul)**: Maps, obstacles, progression.
- **Infra/Backend (Shahriar)**: Repo setup, CI/CD, hosting, optional save system.

---

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript (or TypeScript)
- **Rendering**: Canvas API
- **Version Control**: Git + GitHub
- **Deployment**: Netlify/Vercel (CI/CD)
